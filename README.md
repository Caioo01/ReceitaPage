# 🍽️ Landing Page de Receita

Este foi o meu primeiro projeto utilizando **HTML5** e **CSS3**. O objetivo foi praticar a criação de uma página estática, aplicando conceitos fundamentais de estruturação e estilização de páginas web.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 📚 O que pratiquei

* Estrutura semântica em HTML
* Organização de elementos da página
* Estilização com CSS
* Espaçamentos, cores e tipografia
* Layout de uma landing page

## 💻 Como visualizar

1. Clone este repositório:

   ```bash
   git clone https://github.com/Caioo01/ReceitaPage.git
   ```

2. Abra o arquivo `index.html` no navegador.

## 🎯 Objetivo

Este projeto faz parte dos meus estudos em desenvolvimento web. Apesar de ser simples, representa o início da minha jornada como desenvolvedor e serviu para consolidar os conceitos básicos de HTML e CSS.
